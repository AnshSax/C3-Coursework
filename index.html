Here in my C3 coursework, I will be discussing the advantages and disadvantages of using different Numerical graph solutions. The three methods I will be using are:

  1. Decimal Search (type of systematic search)
  2. Fixed point iteration using Newton-Raphson method
  3. Fixed point iteration after rearranging the equation f(x)=0 into the form x=g(x)

  • Decimal Search Method

    What is decimal search?

    In decimal search, we look for a change in sign in the ‘y’ value of the function. When we find this change in the sign (for example the root lies between ‘x’ values 1 and 2), we go to the next decimal place and divide the intervals into 10 ‘x’ values, i.e.: the list will look like 1.0, 1.1, 1.2, …, 2.0. Again, you find the sign changes, in the ‘y’ values and subdivide that specific ‘x’ interval again into 10 intervals. Do this until you have ‘x’ values which are in 4 significant figures to have a good degree of accuracy.

  • Fixed Point Iteration, Newton-Raphson Method

    Newton- Raphson numerical solution...

    If we have a function ‘f(x)’ which has real roots near an integer, you have to choose that integer for the ‘x0’ value. At this value of ‘x’, we have to find the ‘y’ value on our curve. At this point on the curve, we need to draw a tangent and find where that tangent intersects the x-axis. This ‘x’ value then becomes our ‘x1’ value. For this ‘x’ value, we do the same thing as we did to ‘x0’. The x-intersect of each of the tangents should get us closer and closer to the real root value near that integer.

    To find the value of the next ‘x’ value, we can use the equation? x_(n+1)= x_n- (f(x_n))/(f'(x_n)) This equation can be used as an alternative to the tangent method on the graph. The ‘x’ value should converge to the real root quite quickly using this method.

  • Fixed Point Iteration, Rearrangement Method

    Fixed point iteration-Rearrangement method...

    Rearrangement is my third and final method of numerical solution. In this method, once you come up with a function, you plot the graph for it on ‘Autograph’ programme. Then you also plot the line ‘y=x’. Next you plot function g(x) (which is the original function ‘f(x)’ but just rearranged). Once you have plotted these three lines, choose one real root of your original function. Check if the intersection of the rearranged function and line ‘y=x’ are either directly underneath or directly above this real root. If this is the case, then iterate an integer ‘x’ value (one which is near the root) to get closer and closer to the actual ‘x’ value of the real root you desire. You should find the ‘x’ value to a minimum of 5 significant figures of accuracy.
